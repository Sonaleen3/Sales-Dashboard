# Sales-Dashboard
TITLE : Sales Dasboard 

DESCRIPTION :
The Power BI Sales Dashboard was developed to analyze and visualize sales performance using interactive and dynamic data visualization tools available in Microsoft Power BI. The dashboard provides a comprehensive overview of sales trends, product performance, and regional distribution, helping users make data-driven decisions effectively.

Various Power BI features were used to enhance data exploration and user interaction. Slicers were implemented to allow users to filter the data based on different categories such as region, product, or time period, making the dashboard more flexible and interactive.

Multiple visualization techniques were incorporated to represent the data clearly:

A Donut Chart was used to display the proportion of sales contribution by different product payment type,method

A Stacked Bar Chart was utilized to compare sales by product

A Treemap provided a hierarchical view of sales distribution among different sales by category

A Line Chart was included to illustrate sales trends over time daily sales

The Q&A feature enabled users to ask questions in natural language and instantly generate visual insights from the dataset.

Overall, the dashboard presents complex sales data in an intuitive and visually appealing format, enabling quick analysis, better understanding of key metrics, and improved business decision-making.
